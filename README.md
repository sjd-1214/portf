## ⚡ The "Boss Level" Tech Stack

This project was architected using a modern, mission-critical stack to ensure maximum performance and a premium user experience:

*   **Frontend**: [Next.js 14](https://nextjs.org/) (App Router) for superior performance and SEO.
*   **Animations**: [Framer Motion](https://www.framer.com/motion/) for complex 3D tilts, kinetic physics, and fluid state transitions.
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom **Obsidian & Gold** design system.
*   **Icons/Logic**: [Lucide React](https://lucide.dev/) for precise technical iconography.
*   **Deployment Architecture**: Optimized for Vercel/AWS with an infrastructure-first mindset.

## 🛠️ Installation & Local Development

To inspect the system architecture locally:

```bash
# Clone the repository
git clone https://github.com/sjd-1214/portfolio.git

# Navigate to the control plane
cd portfolio

# Install mission-critical dependencies
npm install

# Initialize local environment
npm run dev
```

The system will be operational at `http://localhost:3000`.
